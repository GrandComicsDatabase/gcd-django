You can do standard issue adds, or add variants by giving the variant name and variant cover status at the end of the issue line. That means, the csv-line needs to have 21 fields (which can be empty), where the last two fields are the variant name and variant cover status. The variant cover status must be either "artwork difference", "only scan difference", or "no difference".

A variant add with status "artwork difference" can optionally be followed by a cover sequence.
